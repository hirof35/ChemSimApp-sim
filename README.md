# 🧪 化学溶液濃度シミュレーター (Chemical Solution Simulator)

Rust と `egui` (eframe) を用いて構築された、リアルタイムな化学溶液（溶質・溶液・溶媒）の動的濃度シミュレーターです。

物理量を変更すると、質量パーセント濃度やモル濃度が即座に再計算され、直感的なインターフェースを通じてインタラクティブにシステムの状態を解析できます。

## 🎯 本プロジェクトの設計思想（構造科学的アプローチ）

本アプリケーションは、単なるUI付きの計算機ではありません。Rustの**強力な型システム**と**メモリ安全性（所有権モデル）**を利用して、化学の本質的な構造をコード上に表現しています。

1. **Newtypeパターンによる概念の厳密な分離**
   溶媒（Solvent）と溶質（Solute）は、内部的には同じ「物質（Substance）」という構造体ですが、システム上では異なる型としてカプセル化されています。これにより、計算論理における誤った混同（バグ）をコンパイル段階で完全に排除します。
2. **所有権のムーブ（移動）による「溶解」の表現**
   物質を溶液システムに投入する際、所有権が溶液（Solution）側へと移動します。これは「物質が液体に溶けて一体化し、独立した存在ではなくなった」という物理的リアリティを模したアーキテクチャです。
3. **即時モードGUI (Immediate Mode GUI) との同期**
   `egui` の「毎フレーム全描画」というリアクティブな特性を活かし、状態（State）の変化が出力（UI）へ一意に決定される関数型的なパイプラインを形成しています。

## ✨ 主な機能

- **インタラクティブなパラメータ調整**: スライダーを動かすことで、水、塩化ナトリウム(NaCl)、スクロース(C₁₂H₂₂O₁₁)の質量（g）および仮想体積（L）をリアルタイムに変更可能。
- **リアルタイム解析・出力**:
  - 溶液の総質量 ($g$)
  - 全体の質量パーセント濃度 ($\%$)
  - 各溶質の物質量 ($mol$)
  - 各溶質のモル濃度 ($mol/L$)
- **物理的限界のフィードバック**: 濃度が特定の閾値（35%）を超えた場合、常温における溶解度の限界を警告するセーフティアラート機能を搭載。
- **マルチプラットフォーム対応 & 日本語豆腐文字対策**: OSのシステムフォント（MS Gothic, Hiraginoなど）を自動検索してランタイムにインジェクション。Windows、macOS、Linux環境での文字化け（□□□）を防止。

🔮 今後の拡張（ロードマップ）
このシステムは構造が完全にカプセル化されているため、以下の機能を容易にプラグイン（拡張）可能です。

温度依存型・溶解度曲線の実装: 温度（℃）スライダーを追加し、物質ごとの溶解度積を超えた場合に「析出量（g）」を動的に計算・描画する。

電離度・pHシミュレーション: 酸・塩基の物質データを追加し、電離度を考慮したpH値の非線形シミュレーションを行う。

マルチスレッド化: 膨大な分子数を扱う熱力学シミュレーション等へスケールアップする際、Rustの Send/Sync 特性を活かして、描画スレッドと計算スレッドを完全に分離。
